# 🎨 DrawGuessRoom — 实时你画我猜（房间版）

一个基于 **Node.js + Socket.IO** 的实时多人“你画我猜”游戏，支持**房间系统**、**观众模式**、**自定义词库**、**别名匹配**、**画笔撤回**等特性。你可以快速部署，与朋友一起享受在线绘画与猜词的乐趣！

---

##  主要特性

-  **房间系统**  
  创建/加入房间，支持 **20 名玩家 + 30 名观众**。

-  **玩家/观众自由切换**  
  可随时在玩家与观众之间切换身份，不影响游戏进程。

   **精准猜词机制**  
  - 自动**去除空格**、**忽略大小写**匹配  
  - 词库支持**别名**（用 `/` 分隔，如 `苹果/红苹果`），回答别名也算正确  

-  **自定义词库**  
  房主可上传 `.txt` 词库文件（逗号分隔），轻松扩展题目。

-  **灵活的游戏设置**  
  绘画时间、选词时间、总轮数均可由房主随时调整。

-  **完整绘图工具**  
  画笔、橡皮擦、填充、撤回、清空，支持颜色选择和笔刷粗细调节。

-  **断线重连**  
  玩家意外掉线后，5 分钟内可重连并恢复历史画布与分数。

-  **实时计分与排名**  
  根据猜词顺序给分，画手根据猜对人数得分，游戏结束展示完整排行榜。

---

##  在线体验

你可以将此项目部署到任何支持 Node.js 的服务器（Railway, Render, VPS 等），或直接在本地运行。

---

##  本地运行

### 环境要求
- Node.js >= 14

### 1. 克隆仓库
```bash
git clone https://github.com/你的用户名/DrawGuessRoom.git
cd DrawGuessRoom
### 2. 安装依赖
bash
npm install
### 3. 启动服务器
bash
node server.js
服务器默认运行在 http://localhost:3000

### 4. 开始游戏
用浏览器打开 http://localhost:3000，输入昵称即可创建或加入房间，邀请朋友一起玩！
